Автоматизированная система «Клавиатурный тренажер» с функциями администратора

Данное приложение предназначено для обучения и совершенствования навыков наборa текста на клавиатуре. Основная цель — развитие навыков слепого десятипальцевого метода печати, увеличение скорости и точности ввода текста.

Начальный экран

После запуска приложения открывается начальный экран. На экране расположены две основные кнопки:

  • Справочная информация — открывает всплывающее окно с информацией о разработчиках программы.
  • Вход — выполняет переход на страницу авторизации пользователя.
Дополнительно на экране отображаются декоративные элементы в виде отдельных кнопок с буквами, имитирующие клавиши клавиатуры. Эти элементы реагируют на соответствующие нажатия клавиш на физической клавиатуре, что наглядно демонстрирует назначение приложения как клавиатурного тренажёра и создаёт эффект «живого» взаимодействия с интерфейсом.

Авторизация и регистрация

С начального экрана при нажатии кнопки "Вход" происходит переход в окно авторизации, предназначенное для входа пользователя в систему. В программе предусмотрено два типа пользователей:

  • Игрок — выполняет упражнения и просматривает личную статистику.
  • Администратор — управляет пользователями и упражнениями, просматривает статистику.

В окне расположены следующие элементы управления:

  • Поле ввода «Логин» — предназначено для ввода имени пользователя. Для администратора есть фиксированный логин.
  • Поле ввода «Пароль» — предназначено для ввода пароля. Для администратора есть фиксированный пароль.
  • Кнопка «Войти» — выполняет попытку входа в систему с введёнными данными.
  • Флажок «Войти как администратор» — определяет режим входа пользователя (игрок или администратор). Должен быть обязательно выбран при вводе логина и пароля администратора, иначе выведется ошибка.
  • Кнопка «Регистрация» — осуществляет переход на страницу регистрации нового пользователя.

Окно регистрации содержит те же поля ввода. Кнопка «Зарегистрироваться» выполняет создание нового пользовательского аккаунта на основе введённых данных.

Вход в качестве игрока

Главное меню игрока

В окне расположены три кнопки:

  • Тренировка — осуществляется преход в окно выбора упражнения.
  • Моя статистика — осуществляется преход в окно статистики.
  • Выход — осуществляется выход из системы и переход к окну авторизации.

Окно выбора упражнения

Сверху расположены кнопки с названием уровня сложности. Повышение уровня сложности происходит слева направо. Всего уровней сложности 5: Новичок, Ученик, Мастер клавиш, Эксперт скорости, Ниндзя.

При выборе уровня сложности появляется список всех существующих упражнений данного уровня. Для каждого упражнения указывается его название и описание в виде количества символов.
Справа снизу также указываются соответствующие уровню сложности: максимум ошибок и максимальное время нажатия.
У каждого упражнения есть кнопка "Начать". При нажатии на неё происходит переход в окно выполнения упражнения.

Слева снизу при нажатии на элемент "В главное меню" осуществляется переход в главное меню пользователя.

Окно выполнения упражнения

Окно выполнения упражнения предназначено для непосредственного тренинга навыков набора текста. Его интерфейс содержит следующие элементы:

  • Контейнер для текста упражнения — расположен по центру экрана. Текст упражнения появляется справа налево, что позволяет пользователю следить за текущим заданием.
  • Панель прогресса — расположена слева от контейнера. Отображает следующие параметры:

    • Прогресс выполнения упражнения
    • Время, затраченное на выполнение
    • Скорость набора
    • Количество сделанных ошибок

  • Флажок «Виртуальная клавиатура» — при активации открывается на экране виртуальная клавиатура. Клавиши виртуальной клавиатуры реагируют на нажатия физических клавиш, что обеспечивает наглядную обратную связь и помогает ориентироваться при наборе текста.
  • Кнопка «Пауза» - останавливает выполнение упражнения.
  • Кнопка «Выход» - осуществляет переход в окно выбора упражнений.

Окно статистики

Окно статистики предназначено для просмотра результатов и анализа прогресса игрока при выполнении упражнений.
Интерфейс окна включает:

  • Панель информации — расположена слева. Содержит сводные данные о прогрессе игрока: общее количество выполненных упражнений, средняя скорость набора, средняя точность, среднее время выполнения.
  • График прогресса — расположен справа. График отображает динамику показателей.

    • По умолчанию график строится по точности.
    • При активации флажка «по скорости» график изменяется, отображая изменения скорости набора.

  • Таблица с историей упражнений — расположена в нижней части окна. Таблица содержит:

    • Названия упражнений
    • Дату их прохождения
    • Характеристики выполнения: скорость набора, точность, количество ошибок

Слева снизу при нажатии на элемент "В главное меню" осуществляется переход в главное меню пользователя.

Окно статистики позволяет игроку визуально отслеживать прогресс, сравнивать результаты и анализировать эффективность тренировки.

Вход в качестве администратора

Главное меню администратора

В окне расположены четыре кнопки:

  • Упражнения — осуществляется преход в окно списка упражнений.
  • Пользователи — осуществляется преход в окно списка пользователей.
  • Статистика — осуществляется преход в окно статистики.
  • Выход — осуществляется выход из системы и переход к окну авторизации.

Окно списка упражнений

Окно списка упражнений предназначено для просмотра, редактирования и управления всеми упражнениями в системе.
Интерфейс окна включает в себя:

  • Кнопка «Создать» — расположена в верхней части окна. При нажатии выполняется переход в окно создания и редактирования упражнения.
  • Таблица всех упражнений — расположена в центральной части окна. Содержит следующую информацию о каждом упражнении:

    • ID упражнения
    • Название
    • Уровень сложности
    • Длина упражнения

  • Кнопки «Редактировать» и «Удалить» — расположены под таблицей.

    • Чтобы изменить или удалить упражнение, необходимо предварительно выбрать его в таблице.
    • Кнопка «Редактировать» выполняет переход в окно создания и редактирования упражнений, где можно изменить данные выбранного упражнения.
  • Слева снизу при нажатии на элемент "В главное меню" осуществляется переход в главное меню администратора.

Окно списка пользователей

Окно списка пользователей предназначено для просмотра и управления статусом пользователей в системе.
Интерфейс окна включает в себя:

  • Поле ввода — расположена в верхней части окна. Используется для поиска пользователя по логину.
  • Кнопка "Поиск" — расположена справа от поля ввода. При нажатии осуществляется поиск по введённому логину.
  • Таблица всех пользователей — расположена в центральной части окна. Содержит следующую информацию о каждом пользователе:
    • Логин
    • Статус
  • Кнопки «Заблокировать», «Разблокировать» и «Удалить» — расположены справа от таблицы.

    • Чтобы изменить статус пользователя или удалить его, необходимо предварительно выбрать его в таблице.
  • Слева снизу при нажатии на элемент "В главное меню" осуществляется переход в главное меню администратора.

Окно создания и редактирования упражнений

Окно предназначено для создания новых упражнений и редактирования существующих.

Интерфейс окна включает:

  • Поле «Название» — для ввода названия упражнения.
  • Поле «Длина» — для указания длины упражнения (количество символов).
  • Выпадающий список «Уровень сложности» — позволяет выбрать сложность упражнения.
  • Флажок «Создать вручную» — при активации появляется дополнительное поле для ввода текста упражнения вручную, вместо автоматической генерации. Количество введенных символов должно соответствовать полю "Длина".
  • Кнопка «Сохранить» - создаёт новое упражнение или сохраняет изменения в существующем.
  • Кнопка «Отменить» — возвращает пользователя в окно списка упражнений без сохранения изменений.

Это окно обеспечивает полный контроль администратора над содержимым тренажёра и позволяет создавать упражнения, соответствующие уровню подготовки пользователей.

Окно статистики

Окно статистики предназначено для анализа результатов пользователей и эффективности упражнений.
Интерфейс окна включает:

  • Панель информации — расположена слева. Содержит сводные данные: общее количество завершённых упражнений, средняя скорость набора, средняя точность, среднее время выполнения.
  • Флажки "По пользователям" и "По упражнениям" - вывод информации на панель и в таблицу в зависимоси от выбранного флажка.
  • График прогресса — расположен справа. График отображает динамику показателей или упражнений.

  • Таблица упражнений — расположена в нижней части окна. Таблица содержит:

    • Уровень сложности
    • Длина
    • Количество прохождений
    • Средняя скорость
    • Средняя точность
    • Среднее время на упражнение
  • Таблица пользователей — расположена в нижней части окна. Таблица содержит:

    • ID пользователя
    • Логин
    • Количество выполненных упражнений
    • Средняя скорость
    • Средняя точность
    • Среднее время на упражнение

Слева снизу при нажатии на элемент "В главное меню" осуществляется переход в главное меню пользователя.
